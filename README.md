<h1>Informacion para importar el script</h1>

<ul>
    <ol>
        Se debe crear una base de datos en postgres con el nombre marketplace
    </ol>
    <ol>
        luego se debe correr el script de importacion de datos en la consola por medio del comando ```psql -U postgres -d marketplace < marcketplace.sql```
    </ol>
    <ol>
        Luego visualizar las tablas en pgADmin :)
    <ol>
</ul>
